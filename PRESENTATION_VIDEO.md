# Script video - OLEEN

Objectif : enregistrer une video claire de 2 a 3 minutes pour presenter le projet.

## Preparation

- Ouvrir le site de presentation dans le navigateur : `docs/index.html`
- Lancer l'application mobile avec `npm start`
- Ouvrir OLEEN dans Expo Go
- Filmer l'ecran du telephone ou utiliser une capture video

## Script court

### 0:00 - 0:15 - Introduction

Image a montrer : site de presentation, section hero OLEEN.

Texte :

Bonjour, je vous presente OLEEN, notre projet d'application mobile realise dans le cadre du lycee. L'objectif est de proposer une application complete qui regroupe plusieurs univers : le sport, l'astronomie, la musique, l'apprentissage du solfege, un reseau social et un module de paris sportifs simules.

### 0:15 - 0:35 - Contexte du projet

Image a montrer : site, section equipe ou modules.

Texte :

Notre equipe est composee de 7 personnes : deux personnes cote developpement, trois designers, une personne pour la communication et une personne pour la partie finances. Mon role est chef developpeur : je m'occupe de la structure technique, du suivi du code et de l'integration des fonctionnalites dans l'application.

### 0:35 - 0:55 - Accueil de l'application

Image a montrer : accueil OLEEN dans Expo Go.

Texte :

Voici l'accueil de l'application. On retrouve un tableau de bord avec les credits virtuels, la progression des cours, les paris en cours et les interactions sociales. L'application est organisee par modules pour que l'utilisateur puisse passer rapidement d'une fonctionnalite a l'autre.

### 0:55 - 1:35 - Les 6 modules

Image a montrer : passer rapidement dans chaque onglet.

Texte :

Le premier module concerne les videos de sports, avec un lecteur visuel et une selection de contenus.
Le deuxieme module est consacre aux sciences astronomiques, avec des contenus educatifs, des evenements du ciel et un quiz.
Le troisieme module gere des paris sportifs simules. Les mises se font uniquement avec des credits virtuels, donc il n'y a pas d'argent reel.
Le quatrieme module permet d'apprendre le solfege par instrument, par exemple piano, guitare, violon ou batterie.
Le cinquieme module presente des performances musicales en direct ou en replay.
Enfin, le dernier module est un reseau social avec profil, publications, likes, commentaires et partages.

### 1:35 - 2:05 - Interaction a montrer

Image a montrer : une action simple dans l'app.

Texte :

Pour montrer que l'application est interactive, je peux par exemple valider un cours de solfege, publier un message dans le reseau social, ou placer une mise virtuelle sur un match fictif. Ces donnees sont sauvegardees localement sur le telephone.

### 2:05 - 2:30 - Technique et livraison

Image a montrer : README GitHub ou site, section technique.

Texte :

Techniquement, OLEEN est developpee avec Expo React Native, ce qui permet de viser Android et iOS. Le projet est versionne sur GitHub, le site de presentation peut etre heberge sur Netlify, et EAS Build est configure pour preparer une version installable de l'application.

### 2:30 - 2:45 - Conclusion

Image a montrer : retour sur l'accueil ou le site.

Texte :

Pour conclure, OLEEN est un prototype mobile complet qui respecte les fonctionnalites demandees. Il montre le travail de notre equipe, l'organisation du projet et une premiere version utilisable de l'application.

## Version tres courte

OLEEN est notre application mobile de projet. Elle regroupe six modules : videos de sports, sciences astronomiques, paris sportifs simules, apprentissage du solfege, musique live et reseau social. L'application est developpee avec Expo React Native pour Android et iOS. Les paris utilisent uniquement des credits virtuels. Le projet est suivi sur GitHub, un site de presentation est pret pour Netlify et EAS Build est configure pour generer une version installable.
